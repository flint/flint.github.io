Flint

Flint is a collection of useful PHP packages. The packages are loosly based around the Symfony and Silex ecosystems.

Packages

  • Flint - Micro framework build on top of Silex. It brings conventions and structure to Silex applications.
  • Luker - Resource tracking. Watch for file changes.
  • Stampie - Mailer for Cloud Providers such as Mandrill, SendGrid, Postmark etc.
  • Antenna - JWT Authentication for Symfony Security based applications.
  • Tacker - Gracefully handle configuration. Based on Symfony Config, mimics most of the features found in Symfony Standard.

All of the packages are used at Peytz & Co for various projects and is therefor battle proven.
